# PKG Hello World

## Pre Requisitos
  -  PKG
  ```sh
    npm install -g pkg
  ```

## Execução
### Clonar e instalar dependencias
```sh
  git clone https://github.com/pereiraadiel/pkg-helloworld
  #
  cd pkg-helloworld
  # instalar dependencias
  yarn
  # ou com npm
  npm i
```

### Gerar o executável
```sh
  pkg .
```

### Executar
```sh
./helloworld
``` 

#### Se tudo der certo, deverá abrir http://localhost:3000 no navegador padrão
